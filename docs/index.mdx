# Weimi-ui

weimi-ui is set of react component that highly customize-able stylings with theme & style props.

- React CSS-in-JSS
- 100% Responsive interface
- Styled-component v4
- Design system theme, with some plugins out of box to import
- Style props support from [Styled System](http://jxnblk.com/styled-system/)


## Getting started

```bash
npm i @weimi/ui
```

```jsx
import { withTheme, GoTheme } from '@weimi/ui/theme'
import { Box } from '@weimi/ui'

const UI = withTheme(GoTheme)

function App() {
  return <UI>
    <Box bg='black' color='white'>Hello world</Box>
  </UI>
}
```
