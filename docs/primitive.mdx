# Primitive

Primitive are component mainly for layout and for other components which are build on top of.

## Base

Base component aollow to pass inline css props anytime

```.jsx
<Base css='background: pink; opacity: 0.5;'>base</Base>
```

<ComponentDoc src={Base} />

## Box

Extends - `Base`

```.jsx
<Box bg='orange' color='white' p={1} width={1}>100% Box</Box>
<Box bg='orange' color='white' p={1} mt={3} width={1/2}>1/2 Box</Box>
<Box bg='orange' color='white' p={1} mt={3} width={1/4}>1/4 Box</Box>
<Box bg='orange' color='white' p={1} mt={3} width={1/8}>1/8 box</Box>
<Box mt={3} mx='auto' p={1} color='gold' fontSize='3'
      width='25rem'
      textAlign='center'
      bg='purple'>25rem Box</Box>
```

<ComponentDoc src={Box} />



## Flex

Responsive flexbox layout component

Extends - `Box`

```.jsx
<Flex alignItems='center'>
  <Flex bg='orange' width={1/4} p='3' mr='2' mt='2' css='align-self: flex-end;'>1</Flex>
  <Flex width={1/4} flexDirection='column-reverse'>
    <Flex bg='orange' p='3' mr='2' mt='2'>1</Flex>
    <Flex bg='orange' p='3' mr='2' mt='2'>2</Flex>
    <Flex bg='orange' p='3' mr='2' mt='2'>3</Flex>
    <Flex bg='orange' p='3' mr='2'>4</Flex>
  </Flex>
  <Flex bg='orange' width={1/4} p='3' ml='2' css='align-self: flex-center;'>1</Flex>
  <Flex bg='orange' width={1/4} p='3' ml='2' css='align-self: flex-start;'>2</Flex>
</Flex>
```

<ComponentDoc src={Flex} />

## AllStyle

Sometimes using `Base` is not enough, `AllStyle` enable you to lookup value from `theme` object.

Use it sparingly for performance concerns.

```.jsx
<AllStyle widht={1} overflow='scroll' height='45px'>
  <ul>
    <li>margin</li>
    <li>margin-top</li>
    <li>margin-buttom</li>
    <li>margin-left</li>
    <li>margin-right</li>
  </ul>
</AllStyle>
```
